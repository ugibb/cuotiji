# 项目初始化文件

## 项目名称

AI 错题集 · 项目制学习平台

## Idea 描述

面向小学生（当前首个场景：华杯奥数备赛），通过 AI 驱动的「项目制学习」体系，帮助学生建立一套面对复杂学习目标的完整方法论：定位 → 目标 → 计划 → 执行 → 复盘。

核心产品模块一：AI 错题集（已开发）

学生拍照上传作业/错题 → AI OCR 识别 + 自动判题 → 基于知识点自动分类 → 遗忘曲线驱动复习计划 → 家长微信报告推送。

核心产品模块二：项目制学习系统（设计中）

- AI 前置沟通评估学生能力层次
- 匹配标准诊断套题 → AI 判题 → 生成个性化能力报告（第一信任节点）
- AI 基于能力画像 × 知识体系 × 备赛周期生成个性化学习计划
- 学生/家长确认计划后，计划与错题集打通，每日任务自动生成
- 里程碑完成后系统自动触发游戏化复盘（成就勋章 + 阶段成就卡）
- 项目结束后生成完整复盘报告

长期愿景：华杯奥数是首个打样项目，成熟后将「项目制学习」框架泛化至其他学习目标（雅思备考、期末冲刺等），最终形成用户自发布学习项目的生态平台。

## 核心痛点

- 错题登记耗时费力，学生敷衍了事
- 学习无目标驱动，随机刷题无法形成闭环
- 不知道自己的薄弱知识点在哪里，不知道该从哪里开始
- 家长无法有效监督，缺乏可信的学习进度反馈
- 备赛准备无系统性规划，临时抱佛脚效果差

## 目标用户

- 主要用户：小学 4-6 年级学生（8-12岁），有竞赛备赛需求
- 买单方：家长（重视素质教育 + 竞赛成绩，对方法论有信任需求）
- 首个场景：华杯小学奥数备赛

## 商业定位

强调「思维能力培养」而非「学科竞赛备考」，契合双减政策语境，同时满足家长对竞赛结果的实际预期。

## 技术栈（tech）

微信小程序（原生 + TypeScript）+ Node.js（Fastify）+ PostgreSQL + 腾讯云 COS/OCR + Claude API
