# AI 错题集 · 项目制学习平台 竞品分析报告

> 报告版本：v2.0
> 更新日期：2026-04-29
> 分析对象：AI 错题集 · 项目制学习平台（微信小程序）
> 目标用户：小学 4-6 年级（8-12 岁），首发场景为华杯小学奥数备赛，买单方为家长

---

## 执行摘要

**核心判断：差异化窗口存在，但护城河需主动构建。**

直接竞品（作业帮、猿题库、学而思）均已具备 AI 拍照、OCR 判题、错题归档等基础能力，但它们的错题功能是「大平台的附属模块」，普遍存在以下三个结构性缺陷：

1. **无项目制思维**：只解决「拍题→存档」，不解决「为什么错、如何系统攻克、阶段性在哪」
2. **无竞赛垂直化**：奥数/华杯题目识别率差、知识点分类体系不匹配竞赛逻辑
3. **家长端是盲区**：家长数据可见度极低，无法辅助家长做备赛计划决策

本产品在「方法论内化」这一维度上的差异化是真实的，但也是最难规模化的。**核心风险不是功能，而是用户习惯养成的成本**——家长和孩子是否愿意将「定位→目标→计划→执行→复盘」这套工作法嵌入日常备赛行为中。

## 一、直接竞品分析：AI 错题 / 作业工具类

### 1.1 猿题库 / 小猿搜题（猿辅导旗下）

**产品概况**

- 月活用户：约 5,000 万（小猿搜题 2024 年数据）
- 主要功能：拍照搜题、AI 解析、错题本（猿题库）、专项练习
- 定价：小猿搜题基础免费；猿辅导在线课 199-599 元/科/学期

**AI 错题能力评估**

| 能力项 | 现状 |
|--------|------|
| OCR 识别 | 强，支持手写题目，汉字+数学公式覆盖广 |
| 知识点分类 | 以教材章节为标准，不适配竞赛知识体系 |
| 遗忘曲线 | 有算法推题，但用户体感「被动接受」，主动复习率低 |
| 家长报告 | 基础学情报告（周报），内容宽泛，缺乏备赛导向 |
| 项目制 / 规划能力 | 无 |
| 奥数 / 竞赛针对性 | 弱，题库有竞赛题但无专项备赛系统 |

**致命弱点**：错题本是「流量入口的附属品」，核心商业逻辑是导流至付费课程，错题习惯养成不是其商业诉求。

---

### 1.2 作业帮

**产品概况**

- 月活用户：约 1.5 亿（2024 年，含直播课用户）
- 主要功能：拍照搜题、AI 讲题（语音/视频）、错题整理、直播课
- 定价：基础免费；直播课 99-398 元/科/学期；YOYO 学习机 2,499-3,299 元

**AI 错题能力评估**

| 能力项 | 现状 |
|--------|------|
| OCR 识别 | 市场领先，数学公式识别率高 |
| 知识点分类 | 细致（教材版本适配），但竞赛题分类体系缺失 |
| 遗忘曲线 | 弱，主要功能是「存档」而非「调度复习」 |
| 家长报告 | 有「家长端」App，但数据颗粒度粗 |
| 项目制 / 规划能力 | 无 |
| 奥数 / 竞赛针对性 | 有竞赛专区，但内容是题库堆砌，非系统备赛 |

**致命弱点**：错题整理功能定位是「答疑的延伸」，用户路径是「搜题→看答案→关掉」，错题本是最末端被访问的功能。

---

### 1.3 学而思（好未来旗下）

**产品概况**

- 月活用户：约 3,000 万（2024年，含学而思网校）
- 主要功能：直播课 + 录播课、AI 自适应学习（学而思 AI 课）、随堂测试错题回顾
- 定价：季度课 1,200-2,800 元；AI 课单独订阅约 199 元/月

**AI 错题能力评估**

| 能力项 | 现状 |
|--------|------|
| OCR 识别 | 课内题目识别良好，依赖课程闭环 |
| 知识点分类 | 强（尤其数学），有竞赛课体系 |
| 遗忘曲线 | 课后「自适应复习」机制较完整，但绑定课程内容 |
| 家长报告 | 较好，有「学情报告」含知识点掌握度 |
| 项目制 / 规划能力 | 有课程学习路径，但是「预设路径」非个性化规划 |
| 奥数 / 竞赛针对性 | 强，有华罗庚金杯、走美杯等专项课 |

**核心威胁**：学而思在竞赛培训上积累深厚，是本产品在「奥数备赛」场景上最直接的内容竞品。但其核心是「卖课」，不是「卖工具」，所以错题管理是为了粘住课程用户，不是独立产品。

### 1.4 近期新兴 AI 错题类小程序 / App（2024-2026）

| 产品名 | 定位 | 核心功能 | 问题 |
|--------|------|----------|------|
| **墨刀错题本**（微信小程序）| 通用错题记录工具 | 拍照存档 + 标签管理 | 工具属性强，无 AI 判题，无习惯系统 |
| **学习强国·错题模块** | 政务导向教育平台 | 配套学习内容的错题归档 | 学科范围有限，无竞赛内容 |
| **妙小程（AI 出题）** | AI 智能出题 | 基于知识点自动生成练习题 | 出题工具，非错题管理 |
| **百度文库·智能练习**（2024 上线）| AI 练习 + 错题追踪 | 文档导入 → AI 出题 → 错题记录 | 使用路径复杂，小学生不友好 |
| **讯飞星火·作业批改** | AI 批改作业 | 拍照批改 + 错题指出 | 面向老师端为主，非学生自用 |

**结论**：2024-2026 年间的新入局者多为「AI 出题」或「AI 批改」方向，真正做「AI 错题集 + 习惯系统」闭环的产品仍未出现成功案例。

## 二、间接竞品分析：竞赛备赛 / 奥数培训类

### 2.1 华杯赛 / 走美杯官方及授权机构

| 机构 / 平台 | 在线产品形态 | 特点 | 局限 |
|-------------|------------|------|------|
| **华杯赛官方** | 无独立 App，官网提供历年真题 PDF | 权威内容，免费 | 无交互、无个性化、纯内容堆砌 |
| **走美杯** | 官网真题下载 + 部分授权机构合作 | 内容权威 | 同上 |
| **学而思竞赛课** | App 内专区（金杯课、走美课）| 系统课程，名师讲解 | 课程贵（500-2000 元/期）、无错题个性化 |
| **高思教育线上** | 线上录播课 + 题库 | 竞赛题库丰富 | 无 AI 个性化，课程体验传统 |
| **思维培优**（各地线下机构）| 多数无线上产品或仅有微信群 | 师资好 | 线上化程度低，无法规模化 |

**机会点**：华杯等竞赛的备赛场景，**缺乏一款能连接「孩子刷错题」和「家长看备赛进度」的轻量工具**。机构提供内容，本产品可以提供「执行层工具」，存在与机构合作的可能性。

### 3.3 家长自建工具现状

| 工具 | 使用场景 | 家长痛点 |
|------|---------|---------|
| **Notion 打卡模板** | 记录每日学习，制定计划 | 需家长手动维护，孩子不参与，可持续性差 |
| **飞书多维表格** | 错题记录表、备赛进度 | 同上，门槛高，操作繁琐 |
| **微信群 / 朋友圈打卡** | 社群监督，晒学习记录 | 有社交压力但无结构化反馈，无 AI |
| **纸质备赛日记** | 传统方式 | 无数据积累，无遗忘曲线 |
| **小红书备赛攻略跟随** | 学习方法参考 | 碎片化，无个性化，执行难度大 |

**结论**：家长自建工具的核心问题是**「有意愿、无工具」**，他们知道需要系统备赛，但市面没有一款轻量、儿童友好、AI 驱动的备赛管理工具。这是本产品最核心的市场缺口。

## 五、竞争格局判断

### 5.1 本产品的差异化优势

**优势一：奥数备赛场景的垂直深度**

没有任何一款数字产品将「错题集」和「奥数备赛项目管理」结合起来，且针对华杯赛这一具体赛事做了知识体系适配。「小而深」的定位使得产品在目标用户中的感知价值远超通用错题工具。

**优势二：方法论内化而非刷题工具**

「定位→目标→计划→执行→复盘」的项目制框架，是将「备赛」当作一个完整学习项目来管理，而非单纯提升刷题效率。这一理念在小学阶段几乎无竞争者，且与家长对「培养学习习惯」的深层需求高度契合。

**优势三：家长端是被主流平台放弃的价值洼地**

猿辅导、作业帮的商业逻辑是「把学生留在平台上」，家长报告是次要的。本产品以「家长买单」为出发点，将家长端体验（备赛仪表盘、错题周报、阶段复盘）作为核心产品力，这是一个结构性差异，无法被大平台快速复制（因为复制后会损害其「让孩子沉浸在平台上」的商业逻辑）。

**优势四：微信小程序的轻量优势**

相比需要下载 App 的主流竞品，小程序的获取门槛极低，尤其适合家长在备赛社群（微信群）中的自然传播。

---

### 5.2 主要威胁来源

**威胁一：学而思的竞赛内容壁垒（最大威胁）**

学而思在竞赛培训领域积累了超过 10 年的内容体系，包括竞赛题库、名师解析、赛事历年真题。如果学而思将其竞赛内容与一个类似本产品的「项目制工具」结合，会形成「内容 + 工具」的完整闭环，对本产品构成降维打击。

**威胁二：大平台的快速模仿**

一旦本产品验证了「奥数备赛 + 项目制 + 家长报告」的商业模式，猿辅导或作业帮可以在 3-6 个月内推出类似功能模块，利用其海量用户基础形成规模优势。

**威胁三：用户习惯养成的高门槛**

项目制学习框架对 8-12 岁儿童有一定认知负荷。如果执行层设计（UI/UX、任务拆解、里程碑设定）不够儿童友好，用户可能在「新鲜感」消失后大量流失，成为另一个「被下载、不使用」的学习 App。

**威胁四：政策合规风险**

国内对面向未成年人的数字产品有严格限制（防沉迷系统、使用时长、数据保护）。竞赛培训类内容也在「双减」政策的模糊地带，需要持续关注监管动向。

---

### 5.3 护城河分析

| 护城河类型 | 评估 | 建设策略 |
|-----------|------|---------|
| **数据护城河** | 中期可建立 | 积累用户错题数据 → 训练垂直于小学奥数的判题模型 → 识别准确率超越通用平台 |
| **用户习惯护城河** | 强，一旦建立 | 打卡连击机制 + 里程碑勋章 → 形成仪式感 → 切换成本高 |
| **内容护城河** | 需主动建设 | 与奥数机构合作引入知识体系 → 或自建华杯知识图谱 |
| **社区护城河** | 中期可建立 | 家长备赛社群 → 口碑传播 → 冷启动关键 |
| **品牌护城河** | 长期目标 | 成为「华杯备赛首选工具」的品类认知 |
| **网络效应** | 弱（单用户工具）| 引入班级 / 社群错题分享功能可增强，但需谨慎避免隐私问题 |

**最重要的护城河判断**：短期内，本产品的护城河是「奥数场景的垂直深度」（大平台不值得做这么细）；中期护城河是「用户习惯 + 历史数据积累」（用了半年的产品很难迁移）；长期护城河是「品类认知」（家长群里提到华杯备赛就想到这个产品）。

## 六、战略建议

### 6.1 差异化坚守点

1. **坚守「方法论内化」而非「刷题效率」**：营销语言、功能设计、用户引导都要围绕「帮孩子学会如何备赛」，而不是「帮孩子做更多题」
2. **坚守「家长是核心用户」的设计视角**：每一个功能都要问「家长如何从中获得决策信心」
3. **坚守「华杯专一」而非「全赛事通吃」**：MVP 阶段不扩展到走美杯、希望杯，专精华杯知识体系

### 6.2 快速验证的关键指标

| 指标 | 目标值 | 验证窗口 |
|------|--------|---------|
