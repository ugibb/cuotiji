## 二、问题：培训班有四个结构性缺陷

### 这是我自己正在经历的事

我的孩子在备战华杯小学奥数。

报了班、花了钱，但每天都在焦虑：孩子今天学的有没有消化？哪些知识点还是空白？离考试还剩多少时间，够不够用？

这些问题，没有老师告诉我答案——不是他们不想，是**培训班的结构决定了他们做不到**。

### 四个结构性缺陷

| 缺陷 | 实际情况 |
|------|---------|
| **诊断是为了分班，不是为了个性化** | 机构有诊断，但目的是判断进快班还是慢班。孩子进了班，不会再为他单独定制学习路径 |
| **课堂节奏统一，跟不上只能转班** | 老师无法同时关照每一个学生。能跟上就跟上，跟不上，只能下一期转慢班——孩子卡在哪个知识点、卡了多久，老师没有机制去知道 |
| **执行无人追踪** | 下课了事，孩子回家有没有复习，没人知道 |
| **家长是盲区** | 过程是黑箱，只能等期末成绩 |

这不是在说老师差。广州优质奥数老师供不应求，家长抢着报名。

但老师的收入来自课时，不是学生成绩。**老师与学生的利益，从结构上就是脱钩的。**

### 用数字说话

| 现状 | 数据 |
|------|------|
| 广州奥数培训均价 | 500 元 / 节课 |
| 暑假备赛费用 | 5,000 元以上 |
| 全年备赛总成本 | 1.5 - 3 万元 |
| 家长能看到什么 | 期末成绩单 |
| 孩子弱在哪个知识点 | 不知道 |

**家长花了大价钱，买到的是「上完课」，不是「学会了」。**

## 三、解决方案：便宜，过程可控，结果有保障，家长全程可见

### 我们是什么

我们不是「便宜版培训班」，也不是「能调教学习习惯的 AI 老师」。

我们的定位是：**为有目标感的备赛家庭，提供结构化执行工具和全程可见的过程管理。**

孩子有备赛意愿，家长参与其中——这是产品发挥作用的前提。在这个前提下，我们解决的是培训班结构上无法解决的四个问题：

| 培训班的缺陷 | 我们的解法 |
|------------|----------|
| 诊断是为了分班 | AI 诊断套题 → 能力报告，精确到每个知识点的薄弱程度，直接驱动个性化学习路径 |
| 课堂节奏统一，跟不上只能转班 | 每个孩子有专属计划，按自己的薄弱图谱推进，不存在「跟不上」的问题 |
| 执行无人追踪 | 每日任务 + 遗忘曲线复习 + 里程碑检验，全程记录 |
| 家长是盲区 | 每日微信日报，今日完成了什么、弱点在哪、距目标多远 |

同时——**价格是培训班的十分之一**。

### 核心产品逻辑：五步闭环

```
定位 → 目标 → 计划 → 执行 → 复盘
```

**定位**：AI 诊断套题生成能力报告——不是通用标签，是具体数据：
> 「你在余数类题目正确率 62%，低于同级均值 23%，这是重点突破区。」

这是产品的**第一信任节点**。家长看到这份报告，是真正相信产品的起点。

**目标**：选定考试日期和目标层级，「距华杯还有 87 天」全程显示，建立心理锚点。

**计划**：AI 基于能力画像生成每日知识点任务和阶段里程碑，家长确认后生效。

**执行**：拍照上传错题 → AI OCR 判题 → 知识点归类 → 遗忘曲线安排复习。每日完成状态实时可见。

**复盘**：里程碑结束系统自动触发：AI 3 问 + 阶段成就卡 + 家长总结报告。

### 和培训班的直接对比

| | 培训班 | 我们的产品 |
|--|--------|----------|
| 诊断用途 | 分班筛选 | 驱动个性化学习路径 |
| 跟不上怎么办 | 下期转慢班 | 按个人节奏推进，无「跟不上」 |
| 执行是否被追踪 | ✗ | ✅ 每日任务全记录 |
| 家长实时可见 | ✗ | ✅ 每日微信日报 |
| 全年费用 | 1.5 - 3 万元 | 499 元 |

## 五、市场

### 双减是双刃剑，但数据站在我们这边

双减政策（2021）是一把双刃剑：短期压缩了整体市场，也压缩了大平台的合规空间。

**真实数据：**

| 年份 | K12 教育科技市场 | 核心驱动 |
|------|----------------|---------|
| 2022 | **-18%**（双减冲击低谷）| 大机构收缩，市场短期萎缩 |
| 2023 | **+22%** | 需求反弹，AI 工具兴起 |
| 2024 | **+27%** | AI 落地加速，个性化工具爆发 |
| 数学竞赛培训（2024）| **+35%** | 逆势增长，需求替代效应显现 |

关键结论：**双减消灭的是大平台的合规空间，没有消灭家长的需求。**

当语数英补课渠道被压缩，有教育焦虑的家长把更多资源集中到竞赛培训——这是「需求替代」，不是「需求消失」。数字工具比线下机构处于更安全的监管位置，同时承接了被挤出的需求。

### 这是一个小而精的市场

我们不打算夸大市场规模：

广州小学生约 20 万，华杯备赛家庭不到 1%——这是一个**很小、很精、高度焦虑、舍得花钱**的用户群。

这正是我们要的第一批用户：**他们有明确的备赛目标，有清晰的截止日期，有过花大钱却感到失控的亲身经历，家长也深度参与其中。**

全国华杯参赛规模约 30-50 万人次/年，有效付费用户约 3-8 万个家庭，早期可寻址 ARR 约 1,500-4,000 万元。

### 扩展路径

```
华杯奥数（当前）
  → 全国奥数 / 小升初冲刺
  → 体育竞技备赛（铁人三项等）
  → 成人技能考试（雅思 / 职业资格）
```

每个新场景复用同一套框架，补充对应知识体系内容，边际成本极低。

我们相信，**「有目标、有截止日期的复杂任务」是一个普遍的人类问题**，华杯是我们选择的第一个打样场景。
