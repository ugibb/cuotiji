Feature 02 · 智能复习卡

每天 5 分钟，
记住更多

根据艾宾浩斯遗忘曲线，自动调度每天的复习题目。翻卡片，标掌握 / 待练习，系统自动更新下次复习时间。

  • CSS 3D 翻转卡片，正面题目 / 背面解析
  • 5 阶段复习间隔：1 / 3 / 7 / 14 / 30 天
  • 每日打卡 + 连击天数激励
  • 3 级 fallback：到期题 → 新题 → 随机题
不是多做题，

Feature 03 · 弱点分析

知道哪里弱，
才能补哪里

基于时间衰减的薄弱章节评分算法，最近犯错权重高，遥远历史权重低。直观展示 Top 5 弱点章节。

  • 时间衰减公式：1 / (1 + daysAgo / 7)
  • 最近一周 / 一月 / 全部 三段时间维度
  • 点击章节直接查看该章所有错题
  • 数据不足 3 题时提示"积累更多错题"
Decay Formula
w=1/(1+d/7)
d = 距今天数，越近权重越高
